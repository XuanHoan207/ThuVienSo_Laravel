## 18. NOTES

### Authentication:
- Sử dụng Laravel Breeze hoặc Fortify làm base
- Middleware 'auth' cho protected routes
- Session-based cart hoặc database cart

### Database Design:
- users: id, name, email, password, avatar, points, role, timestamps
- categories: id, name, slug, description, parent_id
- books: id, title, slug, description, price, category_id, user_id, status, timestamps
- authors: id, name, slug, bio, avatar
- book_author: book_id, author_id, role
- tags: id, name, slug
- book_tag: book_id, tag_id
- reviews: id, user_id, book_id, rating, comment, status, timestamps
- comments: id, user_id, book_id, parent_id, content, timestamps
- wishlists: user_id, book_id
- purchases: user_id, book_id, points_spent
- downloads: user_id, book_id, downloaded_at
- notifications: id, user_id, type, title, content, read_at, timestamps
- point_transactions: id, user_id, type, amount, description, timestamps
- carts: user_id, book_id, quantity
- contacts: id, name, email, subject, message, timestamps
- sliders: id, image, title, link, status
- coupons: id, code, type, value, min_purchase, expires_at

### API Endpoints (cho AJAX):
- GET /api/books - List books với filters
- GET /api/books/{slug} - Book detail
- POST /api/cart/add
- DELETE /api/cart/{id}
- POST /api/wishlist/toggle
- POST /api/reviews
- POST /api/comments
- GET /api/notifications
- PATCH /api/notifications/{id}/read
- GET /api/authors
- GET /api/author/{slug}/books
